# Button按钮
常用的操作按钮。

## 基础用法
基础的按钮用法

:::demo
```html
<template>
  <div>
    <hButton>默认按钮</hButton>
    <hButton type="primary">主要按钮</hButton>
    <hButton type="success">成功按钮</hButton>
    <hButton type="info">信息按钮</hButton>
    <hButton type="warning">警告按钮</hButton>
    <hButton type="error">错误按钮</hButton>
  </div>
  <div style="margin-top: 10px">
    <hButton outline>默认按钮</hButton>
    <hButton outline type="primary">主要按钮</hButton>
    <hButton outline type="success">成功按钮</hButton>
    <hButton outline type="info">信息按钮</hButton>
    <hButton outline type="warning">警告按钮</hButton>
    <hButton outline type="error">错误按钮</hButton>
  </div>
</template>
```
